## Program Hello World

```ruby

	"Halo #{nama}"
end
```

```py
def halo(nama):
	print ' '.join(['Halo', nama])
```

## Penggaya dalam Markdown

Italic dengan `*italic*` &rarr; *italic*

Bold dengan `**bold**` &rarr; **bold**

List tak berurut:

```text
- List 1
- List 2
  - List 2.1
    - List 2.1.2
- List 3

```

Output:

- List 1
- List 2
   - List 2.1
      - List 2.1.2
- List 3


List berurut:

```text
1. List 1
2. List 2
   - List 2.1
     - List 2.1.2
3 List 3
```

Output:

1. List 1
2. List 2
   - List 2.1
     - List 2.1.2
3 List 3

```text
templates/static/css/buku.css
```

```css
ol {
	list-style: decimal
}
```
